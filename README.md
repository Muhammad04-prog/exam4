# WINDOM - Строительная компания (Казань)

Сайт строительной компании WINDOM. Занимаемся проектированием и строительством частных домов, коттеджных поселков и таунхаусов в Казани и Татарстане.

### Технологический стек
*   **Frontend**: React 19 + Vite 6
*   **UI**: Material UI (MUI) v6
*   **Анимации**: Framer Motion
*   **Слайдеры**: Swiper.js
*   **Локализация**: i18next
*   **API**: Axios (загрузка данных из db.json)
*   **Роутинг**: React Router v7
*   **Стили**: MUI sx prop + Tailwind CSS (v4)

### Особенности проекта
*   **Каталог объектов**: Динамический список домов с фильтрами по этажности и типу (таунхаусы).
*   **Карточки домов**: Детальное описание материалов, конструктивных особенностей и инженерных систем.
*   **Калькулятор**: Расчет ипотеки с учетом текущих ставок.
*   **RU/EN**: Поддержка двух языков.
*   **UX/UI**: Плавные переходы страниц, адаптив под все экраны и современный дизайн.

### Как запустить
1. `npm install` — установка зависимостей.
2. `npm run dev` — запуск локально.
3. `npm run build` — сборка проекта.

### Структура
*   `src/components` — общие блоки (шапка, футер, секции).
*   `src/pages` — основные экраны приложения.
*   `src/locales` — переводы интерфейса.
*   `public/db.json` — данные по домам (мок-бэкенд).
